# VRG_res
VRG interview application project in c++


# build
``sh mkdir build``
``sh cd build``
``sh cmake .. -DCMAKE_TOOLCHAIN_FILE=/home/name/vcpkg/scripts/buildsystems/vcpkg.cmake``
``sh make``
`./main`

# clean
``sh make clean``

# requirements
- c++23 >=
- CMake 3.22 >=
- EnTT (https://github.com/skypjack/entt)
